# Unity WebGL 2D Platformer (Personal Project)

**Live demo:** https://zimzamyoyoman.github.io/TheRickAndMortyPlatformer/  
**Repository:** https://github.com/zimzamyoyoman/TheRickAndMortyPlatformer

A Unity WebGL 2D platformer fan project (Rick & Morty themed). You play as Morty, collect crystals, and avoid hazards/obstacles.

---

## What’s in this repo (important)
This repository hosts the **Unity WebGL build output**, published via GitHub Pages. It contains:
- `index.html`
- `Build/`
- `TemplateData/`

**Note:** The Unity source project (Assets/ProjectSettings/C# scripts) is **not included** (it’s no longer available). This repo is intended as a **playable demo + deployment reference**.

---

## Play the game
Open the live demo link above in a desktop browser.
